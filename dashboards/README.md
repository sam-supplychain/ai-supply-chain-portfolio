# Supply Chain Dashboards

## Purpose
Dashboards convert raw supply chain data into actionable insights for planning, execution, and performance tracking. They enable faster, data-driven decision-making.

## Key Metrics & KPIs
- Forecast accuracy & bias
- Inventory turns & working capital
- Supplier OTD (On-Time Delivery)
- OTIF / fill rate
- Production capacity utilization

## Tools
- Power BI
- Excel
- SQL extracts
- Python for preprocessing (optional)

## Example Dashboards
- Demand forecast vs actuals
- Inventory health (excess, shortage)
- Supplier performance scorecard
- S&OP executive summary
